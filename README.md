## CLI Commands

``` bash
# install dependencies
npm install

# serve with snowpack localhost:8080
npm run start

# serve with preact-cli at localhost:8080
npm run dev

# build for production with minification
npm run build

# test the production build locally
npm run serve

# run tests with jest and enzyme
npm run test
```
